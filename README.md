> 使用若依plus版本集成chatgpt-前端-后台管理

> 本项目完全开源免费！
后台管理界面使用elementUI服务端使用Java17+SpringBoot3.X

实现功能：
1. 微信用户免登录
2. 消息历史记录
3. GPT-3.5对话模型
4. 敏感词过滤
>测试功能：
>AI绘画
